# 基本语法

| 元素     | Markdown语法                                         |
| -------- | ---------------------------------------------------- |
| 标题     | # h1  <br />## h2<br />### h3                        |
| 粗体     | \*\*bold text\*\*                                    |
| 斜体     | \*italicaized text\*                                 |
| 块引用   | > blockquote                                         |
| 有序列表 | 1. First item<br />2. Second item<br />3. Third item |
| 无序列表 | - First item<br />- Second item<br />- Third item    |
| 代码     | ``                                                   |
| 水平线   | ---                                                  |
| 超链接   | \[ title ]\(https://www.example.com)                 |
| 图片     | !\[ alt text ](image .jpg)                           |

> 更多详情 -> [Markdown基本语法](http://markdown.p2hp.com/basic-syntax/)
